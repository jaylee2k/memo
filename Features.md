# Features

## Current Features
- 메모 그룹 계층 트리(생성/수정/삭제, 하위 포함 soft delete).
- 그룹별 메모 목록/상세 편집(제목, Markdown 내용, 폰트 속성, 알람 설정).
- 메모 제목 검색/필터(실시간).
- 메모 자동 저장(디바운스 500ms, 유효성 오류 시 저장 차단).
- 메모 삭제/다른 그룹 이동.
- 휴지통(복구, 선택 영구삭제, 비우기, 90일 경과 정리).
- Sticky Note 다중 독립 창(창 위치/크기 저장, 커스텀 다크 UI).
- 알람(토스트 표시, 스누즈 5/10/30, 반복: 일/주/월 + 종료일).
- Markdown 미리보기(WebBrowser + Markdig).
- 트레이 아이콘(클릭: 메모 관리함, 더블클릭: 환경설정).
- 환경설정(전역 기본 폰트 설정, validation 포함).
- SQLite 로컬 저장 + 시작 시 스키마/기본 데이터 보장.

## Planned Features
- 메모 본문/태그 기반 고급 검색.
- Sticky Note 하단 툴바 버튼의 실제 서식 동작 연결.
- 반복 알람 정밀 스케줄링(기준 시각 유지, DST 고려).
- 통합/시나리오 테스트 확장(서비스 + DB 경계 테스트).
- 설치 파일 코드서명 및 릴리스 자동화 고도화.
