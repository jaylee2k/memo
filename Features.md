# Features

## Current Features
- 메모 그룹 계층 트리(생성/수정/삭제, 하위 포함 soft delete).
- 그룹별 메모 목록/상세 편집(제목, Markdown 내용, 폰트 속성, 알람 설정).
- 메모 자동 저장(디바운스 500ms).
- 메모 삭제/다른 그룹 이동.
- 휴지통(복구, 선택 영구삭제, 비우기, 90일 경과 정리).
- Sticky Note 다중 독립 창(AlwaysOnTop, 창 위치/크기 저장).
- 알람(토스트 표시, 스누즈 5/10/30, 반복: 일/주/월 + 종료일).
- Markdown 미리보기(실시간 HTML 렌더링).
- 트레이 아이콘(클릭: 메모 관리함, 더블클릭: 환경설정).
- 환경설정(전역 기본 폰트 설정).
- SQLite 로컬 저장 + 시작 시 스키마/기본 데이터 보장.

## Planned Features
- Markdown 고급 렌더링 옵션(테마, 코드 하이라이트).
- 통합/시나리오 테스트 확장(서비스 + DB 경계 테스트).
- 설치 파일 코드서명 및 릴리스 버전 자동화 고도화.
- 메모 검색/필터(제목/내용/알람 상태).
